В проекте созданы 2 класса, представляющие продукты (Product) и категории (Category).

В классе Product описаны атрибуты name, description, price и quantity, 
а в классе Category - name, description и products. 
В последнем также добавлены 2 атрибута класса: количество категорий (category_count) 
и количество товаров (product_count).

Если создаётся товар, количество на складе которого равно 0, возникает ошибка ValueError и 
печатается сообщение "Товар с нулевым количеством не может быть добавлен".

Для выделения категорий товаров от класса Product созданы классы-наследники: 
Smartphone (для представления смартфонов) и LawnGrass (для представления газонной травы).

Создан базовый абстрактный класс BaseProduct (родительский для класса продуктов) 
и класс-миксин, который печатает в консоль информацию о том, 
от какого класса и с какими параметрами был создан объект.

Для всех классов добавлена инициализация так, чтобы каждый параметр 
был передан при создании объекта и сохранён.

Для классов написаны тесты, которые проверяют корректность инициализации объектов 
в этих классах, а также подсчёт количества продуктов и категорий.
